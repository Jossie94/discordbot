# P4 DiscordBot Project

## Features
- Customize commands
- Logging of channels
- Easy to use GUI

## Using
What we use:
- [Discord] - As the target for the bot.
- [DiscordJS] - The code library for the bot.
- [NodeJS] - The server and package manager.
- [Jetbrains] - Built with Jetbrains' IDE's
